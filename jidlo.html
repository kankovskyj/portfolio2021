Landing page redesign

Identifying the pain points

When I was tasked with a redesign, the client wanted something simply better looking. I had to convince them, that user research had to be done before we even start with designing. And after going through google analytics, heatmaps and user testing, I was confident to start the actual process.

Solving problems with design

After my analysis, I identified the main problems - the handwriting font in the header does not work well in smaller font sizes rendering the top navigation almost useless, main action button is very hard to find on the page overall, the background image clutters space and shows bread, which does not suit the healthy-food model of the client.

The result

What I presented the client was an overall much more user friendly solution that stayed true to the initial design. I helped test the new version and result overwhelmingly exceeded the clients expectations. The site is now being developed by a software studio.
